## 一、技术架构设计方案

### 1.1 设计思想

**核心理念：组件化 + 状态驱动 + 渲染分离**

AgentUI 采用「数据驱动渲染」的架构思想，将业务逻辑（Agent 状态管理、任务调度）与视觉渲染（PixiJS 像素地图、角色动画）彻底解耦。前端作为纯展示层，通过 HTTP 轮询从 Flask 后端获取 Agent 状态，再由 Pinia Store 驱动 PixiJS Canvas 重绘。

**关键设计决策：**

1. **双层渲染架构**：Vue3 负责 UI 面板（状态栏、对话框、侧边栏），PixiJS 8 负责像素地图和角色渲染。两者通过 Pinia Store 共享状态，互不干扰。
2. **轮询而非 WebSocket**：2-3 秒 HTTP 轮询，降低实现复杂度，适合 9 个 Agent 的低频状态更新场景。
3. **Spritesheet 统一管理**：所有像素素材通过 PixiJS Assets 统一加载，使用 TextureAtlas 管理角色动画帧，避免运行时频繁请求资源。
4. **功能区坐标系统**：地图采用绝对像素坐标 + 逻辑区域映射，4 个功能区各有独立的坐标范围和交互逻辑。
5. **Agent 状态机**：每个 Agent 拥有 `idle / working / meeting / resting / debugging` 五种状态，状态变化驱动角色位置和动画切换。

### 1.5 Agent 状态机

```
         ┌──────── idle ────────┐
         │          │           │
         ▼          ▼           ▼
      working    meeting    resting
         │          │           │
         ▼          │           │
     debugging ◄────┘───────────┘
         │
         ▼
       idle
```

状态与位置映射：
- `idle` → 自己工位（默认）
- `working` → 自己工位（播放打字动画）
- `meeting` → 会议区（走到圆桌旁）
- `resting` → 休息区（走到沙发旁）
- `debugging` → Bug 角落（走到角落，头顶冒烟特效）

## 二、模块职责说明

### 2.1 API 层 (`src/api/`)
- 封装 Axios 实例，baseURL 指向 `http://localhost:19000`
- 统一错误处理和响应拦截
- 为每个后端接口提供类型安全的调用函数

### 2.2 Store 层 (`src/stores/`)
- `agentStore`：管理 9 个 Agent 的实时状态（位置、动作、任务、对话）
- `officeStore`：管理办公室布局配置（功能区坐标、工位分配）
- `pollingStore`：管理轮询生命周期（启停、频率、健康状态）

### 2.3 PixiJS 渲染层 (`src/pixi/`)
- `PixiApp`：初始化 PixiJS Application，挂载到 Vue 容器
- `OfficeMap`：渲染地板瓦片、墙壁、家具等静态元素
- `AgentSprite`：每个 Agent 的像素角色，支持行走动画和状态动画
- `ZoneRenderer`：渲染 4 个功能区的视觉边界和高亮效果
- `PathFinder`：A* 寻路，计算 Agent 从当前位置到目标区域的行走路径
- `CameraController`：鼠标拖拽平移、滚轮缩放、边缘滚动

### 2.4 Vue UI 层 (`src/components/`)
- 覆盖在 PixiJS Canvas 之上的 HTML/CSS UI
- 提供 Agent 详情面板、任务列表、对话气泡等交互界面
- 通过 CSS `pointer-events` 控制穿透，不阻挡地图交互

## 三、风险项

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| PixiJS 8 与 Vue3 生命周期冲突 | Canvas 销毁/重建时内存泄漏 | 在 `onUnmounted` 中严格调用 `app.destroy(true)` |
| 像素素材版权不明确 | 法律风险 | 仅使用 CC0/CC-BY 授权的 LPC、Kenney 素材 |
| 轮询频率过高导致后端压力 | 接口超时 | 自适应降频 + 请求去重 |
| Spritesheet 加载耗时 | 首屏白屏 | 资源预加载 + Loading 进度条 |
| Flask 无 CORS 头 | 跨域请求被拒 | 后端添加 flask-cors |
| A* 寻路在大地图上性能差 | 角色移动卡顿 | 预计算路径缓存 + 简化网格 |
